--- layout: article title: 不安装Oracle也可以使用pl/sql developer(解决pl/sql developer连接64位Oracle) categories: [database] tags: [instantclient,pl/sql developer,oracle] date: 2013-01-22 ---

pl/sql developer一般用来连接oracle，是一款比较好的工具，但是为了使用它都要在本机安装那笨重的oracle，其实不用那么麻烦，oracle提供了客户端，而且还是免安装的，到oracle官网下载下来，然后在pl/sql developer上配置下即可。

第一步:进入oracle官网,找到Downloads下的Oracle Instant Client

可以看到有不同系统版本的,你只要下载对应的版本即可(需要登录)，

PS：下载64位的windows客户端，就可以使pl/sql developer连接64位Oracle，我下载的是32位的

下载好 instantclient-basic-nt-12.1.0.1.0.zip 后，解压文件到任意路径，然后打开pl/sql developer，打开工具-》首选项（TOOLS->PERFERMANCE），找到红色标记处

在上面填上你刚刚下载的文件解压后的oci.dll文件的路径（E:\tools\instantclient-basic-nt\oci.dll）

保存，重新启动pl/sql developer，再启动之前，在E:\tools\instantclient-basic-nt\路径下建立文件夹NETWORK\ADMIN，在ADMIN文件夹下建立tnsnames.ora文件，添加数据库的TNS信息,模板如下

localhost =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = TCP)(HOST = 127.0.0.1)(PORT = 1521))
    )
    (CONNECT_DATA =
    (SERVICE_NAME = orcl)
  )
)

还有最后一步，设置一个环境变量，不然中文都是问号

名称:NLS_LANG
值:SIMPLIFIED CHINESE_CHINA.ZHS16GBK
